PocketDoc Medical Application

Addressing the Healthcare Access Crisis

Inspired by a personal experience with long wait times and unreliable medical information online, PocketDoc aims to provide reliable medical guidance and reduce burden on overtaxed healthcare systems.

Visit app.pocketdoc.me

Technical Solution

Full-stack mobile application using Flutter with a machine learning model that asks dynamic follow-up questions based on reported symptoms to provide accurate guidance.

  • Cross-platform mobile development with Flutter and Dart
  • Backend system built on Firebase
  • Intuitive symptom reporting interface

Key Challenges & Solutions

  • Cost Reduction: Replaced third-party service costing $10,000/month with custom ML model, reducing cost to $0.04 per diagnosis
  • Regulatory Adaptation: Pivoted to clinic workflow enhancement with intelligent patient onboarding

Technical Innovations

  • Custom ML model based on LLMs to simulate doctor's diagnostic approach
  • Proprietary dataset developed through hundreds of test runs
  • Cost-efficient architecture that maintains performance quality

Key Learnings

  • Mobile app development with Flutter ecosystem
  • ML model training and healthcare application integration
  • Healthcare regulations and EMR/EHR systems
  • Adaptability in overcoming technical and regulatory challenges
